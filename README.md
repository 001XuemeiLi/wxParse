## wxParse-微信小程序富文本解析组件
**支持Html及markdown转wxml可视化**
* 版本号:0.3
* 来源: [微信小程序开发论坛-weappdev](https://weappdev.com)
* 作用: 将Html/Markdown转换为微信小程序的可视化方案


## 基础数据格式

```
parsedata:{
    view:{},//样式存储
    nodes:{},//展示需要的存储节点
    images:[],//存放图片对象数组
    imageUrls:[],//存放图片url数组
}

```
